Day 04 - Monday, January 04, 2021

2021-01-04

Today I decided to make a simple Chrome Extension and also going through Android Java Course that I started on the first day of the year.

What I Learned Today

💻 Programming

  • Revised Chrome Extension Basics - I wanted to make a simple Chrome Extension that just plays and audio file from the provided array of names of each audio file. I already knew how to program a Chrome Extension because I have programmed some other simple chrome extensions before for my personal uses like one chrome extension that does not let me uses Reddit for more than 15 minutes in 4 hours. If anyone wants to learn to program a Chrome Extension I highly recommends Daniel Shiffman's Blog Post and Youtube Playlist. Daniel covers almost everthing you need to start working with Chrome Extensions but if anyone wants to go deeper into Chrome Extensions and want to build a complex Chrome Extension Official Documentation will help alot.
  • Javascript Audio() Obejct to work with audio on the web. This was the first time I used this Object in any of my projects.

🗾 Language[日本語]

  • 介護士 (かいごし) means Care Worker. Kanji used are Jammed In, Safegaurd and Gentlemen.
  • すき焼き (すきやき) Sukiyaki, a Japanese dish prepared and served in a Japanese Hot Pot style. kanji used means bake.
  • 新幹線 (しんかんせん) means Bullet Train.
  • 故障 (こしょう) means break, like breaking something.
  • 修理 (しゅうり) means fix, like fixing something. The first kanji is for discipline and second one for logic. So the logic is you are discipling something.
  • 郵便局 (ゆうびんきょく) Post Office.
