This is the first time I am doing bloging just for the sake of learning more and more everyday and doing more stuff and be more produtive so I don't really know what kind of blog I should do or what would be better that is why I write all I learn in the day and while I am learning I will try to improve my writing and also find new ways to do blogging in a better way than the previous day.

When I started this blog from gatsby-starter-blog there were already other starter kits on the Gatsby website but I choose this because first it will be easy to learn and maintain it if I build most of it myself and second there will be too much functionality and plugins in the other kits that I Know I will not be needing.

What I Learned Today

💻 Programming

  • PWA - Progressive Web Apps are Installable WebApps that work on all Platforms (Android, iOS, PC) are built with basic web technologies like HTML, CSS and JavaScript. This Blog Post explains in detail what makes PWAs are the future of the internet.
  • WebApp manifests - In addition to learning about PWAs I learned about WebManifests file which provides information about a webapp in a JSON file, necessary for the web app to be downloaded and be presented to the user similarly to a native app.
  • SASS - CSS Preprocessor - I already have worked with SASS before but I didn't learned all of the SASS syntax before and for this blog project I am again using it but I will try to use as much of it functionality like Mixins, Modules and Partials that I didn't used before in any project.
